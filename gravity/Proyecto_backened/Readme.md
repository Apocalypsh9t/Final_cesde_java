Este proyecto es una API REST desarrollada con Spring Boot orientada a la gestión de usuarios, materiales e historial dentro del sistema. La aplicación permite realizar operaciones como registro de usuarios, inicio de sesión, consulta de información, creación y actualización de datos mediante diferentes endpoints HTTP.

El proyecto fue desarrollado utilizando Java, Maven y Swagger/OpenAPI para la documentación interactiva de las APIs. Además, implementa arquitectura por capas utilizando controllers, services, repositories y DTOs para organizar mejor la lógica de negocio y el manejo de datos.

La documentación de los endpoints puede visualizarse desde Swagger una vez ejecutado el proyecto.


Cómo ejecutar el proyecto

Clonar el repositorio:


git clone https://github.com/Apocalypsh9t/Final_cesde_java.git


Entrar a la carpeta del proyecto:


cd Final_cesde_java/gravity/Proyecto_backened


Ejecutar el proyecto:

.\mvnw.cmd spring-boot:run

 Swagger

La documentación de la API se encuentra disponible en:


http://localhost:8080/swagger-ui/index.html
